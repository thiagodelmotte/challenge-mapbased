# challenge-mapbased

iOS App

## Requirements

- Swift 4+
- CocoaPods
- Mapbox
- SwiftyDrop

### CocoaPods

[CocoaPods](http://cocoapods.org) is a dependency manager for Cocoa projects. You can install it with the following command:

```bash
$ gem install cocoapods
```

After that, go to project directory and run:

```bash
$ pod install
```
